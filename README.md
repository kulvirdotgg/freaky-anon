# `anon-gets-freaky`

**Anonymous** and **ephemeral** chat room which self destructs are specified amount of time.
If anons gets too freaky any user can self destruct the room and it will destroy the room for everyone.

---

## Tech stack

- [Typescript](https://www.typescriptlang.org/)
- [Next.js](https://nextjs.org/)
- [Elysia JS](https://elysiajs.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ShadCN](https://ui.shadcn.com/)
- [Biome](https://biomejs.com/)
